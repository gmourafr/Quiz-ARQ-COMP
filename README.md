# 🖥️ Quiz ARQ COMP

Um quiz básico desenvolvido para revisar e estudar os conceitos de Arquitetura de Computadores de forma simples e prática.

## 📚 Conteúdos abordados

- CPU
- ULA
- Registradores
- Memórias (RAM, ROM, Flash, etc.)
- DMA
- Barramentos
- Processadores Intel
- Dual Core e Quad Core

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎯 Objetivo do projeto

O projeto foi criado como atividade acadêmica para auxiliar na revisão dos principais conceitos de Arquitetura de Computadores através de perguntas interativas.
